
Run XAMPP in administrator for the best results 

![](../assets/Pasted%20image%2020260718152421.png)
You can auto-config the XAMPP control panel to auto start Apache and MySQL when launching 

The websites files and folder can be found inside the folder created by XAMPP -> C:\xampp\htdocs

Access the webpage by starting Apache and MySQL, going to a web browser and typing -> \http:\\localhost
